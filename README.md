# game-design
Jogo destinado a disciplina de game design. Utilizamos apenas python.
>
# Execução
**Certifique-se que o python está instalado na sua máquina** <br>
Depois basta executar o comando para que a biblioteca pygame seja instalada pela pip:
```
python -m pip install pygame
```
Pronto, agora basta digitar:
```
python jogo.py
```
